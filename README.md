## GenAI Projects

### [production-adk-agent-google](https://github.com/MvMukesh/production-adk-agent-google)

Production **claims-assistant** on Google ADK — custom tools, persistent sessions, Next.js adjuster dashboard, Cloud Run deployment.

**Impact:** Faster FNOL draft generation · consistent policy-grounded answers · full tool-trace audit path  
**Stack:** Google ADK · Gemini · Tavily MCP · Next.js · Docker · GCP Cloud Run

### [LangMem-Insurance-Claims-Copilot](https://github.com/MvMukesh/LangMem-Insurance-Claims-Copilot-)

Human-in-the-loop **insurance FNOL copilot** — LangChain agents, LangMem, Chroma RAG, FastAPI, Next.js.

**Impact:** Adjuster-assisted triage · coverage recommendations grounded in policy KB · safe rollout by design  
**Stack:** LangChain · LangMem · Chroma · FastAPI · Next.js
